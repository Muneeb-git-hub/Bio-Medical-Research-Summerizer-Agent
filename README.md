# Bio-Medical-Research-Summerizer-Agent
 Developed an end-to-end AI agent to automate biomedical literature review, featuring instant PDF summarization and 
integrated search via the Semantic Scholar API. 
▪ Engineered a targeted Q&A extraction module that provides precise answers from complex methodology and results 
sections, backed by confidence scoring. 
▪ Implemented NLP-driven trend analysis to visualize keyword frequencies, alongside a custom novelty detection algorithm 
that scores and highlights unique research contributions.
